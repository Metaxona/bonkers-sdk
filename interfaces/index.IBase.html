upgradeToAndCall

  • upgradeToAndCall(newImplementationAddress, params?): Promise<ContractInteractionReturnType<`0x${string}`>>

  • Method used to upgrade (Controller|Vault|VaultFactory) contract in an unsafe manner since there will be no storage layout check during upgrade

    it is only recommended to use this method when upgrading to official verified implementations to make sure no storage collisions will happen which would overwrite the contract storage values causing data to be corrupted rendering the contract unusable and or exploitable

    Parameters
